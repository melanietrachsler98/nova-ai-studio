# Nova AI Studio 🤖

## Descripción del proyecto

Nova AI Studio es un sitio web institucional desarrollado para una empresa ficticia enfocada en soluciones de inteligencia artificial y automatización de procesos para negocios modernos.

El objetivo del proyecto es presentar los servicios, soluciones y propuesta de valor de la marca mediante una experiencia digital profesional, clara y adaptable a diferentes dispositivos.

El sitio fue diseñado con una identidad visual propia, priorizando una estética moderna, minimalista y tecnológica que representa el concepto de innovación asociado a la inteligencia artificial.

---

## Tecnologías utilizadas

- HTML5
- SASS / SCSS
- CSS3
- Bootstrap
- JavaScript
- Git y GitHub
- Vercel

---

## Características principales

- Diseño web institucional con 5 páginas:
  - Inicio
  - Servicios
  - Soluciones
  - Nosotros
  - Contacto

- Diseño responsive adaptado para:
  - Desktop
  - Tablet
  - Mobile

- Arquitectura de estilos con SASS:
  - Variables
  - Parciales
  - Mixins
  - Nesting
  - Extend

- Integración de Bootstrap adaptada a la identidad visual del proyecto.

- Optimización de imágenes y recursos para mejorar rendimiento.

- Implementación de buenas prácticas de HTML semántico y SEO básico.
